# 리나 BODY LAB

단일 HTML 파일로 동작하는 개인 운동/식단/컨디션 기록 앱입니다.

## 사용법
1. `index.html`을 브라우저에서 엽니다.
2. 날짜를 선택하고 운동, 단백질, 컨디션, 메모를 입력합니다.
3. 저장하면 브라우저 localStorage에 기록됩니다.
4. 데이터 백업 버튼으로 JSON 파일을 내려받을 수 있습니다.

## v5 업데이트
- 실내자전거/천국의계단 시간 입력
- 브릿지/복근/어깨·폼롤러 횟수 입력
- 호흡 연습 기록
- 아침/점심/저녁 단백질 체크
- 컨디션 1~5점
- 인바디 메모
- 달력 운동 완료 표시
- 최근 기록 목록
- 데이터 백업/불러오기
